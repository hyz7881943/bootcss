Metro UI CSS 是一套用来创建类似于Windows 8 Metro UI风格网站的样式. 这组风格被开发成一个独立的解决方案。

Metro UI CSS包含两种类型的许可证: MIT 和 Commercial

LESS

Metro UI CSS是用LESS在开发. 一个很优秀的男人，Alexis Sellier创建了这个动态CSS语言LESS。 它使开发CSS更快、更简便、更有趣。
